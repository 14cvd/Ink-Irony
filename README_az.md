# Ink & Irony (Vəhşi Eskiz Dəftəri) 🖋️📓

**Ink & Irony**, tamamilə **SwiftUI** və **SwiftData** ilə yaradılmış premium, mükəmməl dizayna sahib iOS söz tapma (Adam Asdı tipli) oyunudur. Tətbiq gözəl "əllə çəkilmiş eskiz" estetikası, titrəyən mürəkkəb animasiyaları, xüsusi toxunma hissləri (haptics) və hər səhvinizə reaksiya verən təhqiramiz "offline AI" sistemi ilə fərqlənir.

[**App Store-dan Endir**](https://apps.apple.com/app/id6760973464)

---

## 🎨 Xüsusiyyətlər

### 1. Əllə Çəkilmiş Eskiz Estetikası
- **Dinamik UI:** Xüsusi `Shape` animasiyaları əsl qələm çəkilişi effektini yaradır.
- **Tema Sistemi:** "Aged Parchment" (Açıq) və "Charcoal Sketch" (Qaranlıq) modlarını dəstəkləyən `ThemeManager`.
- **Xüsusi Tipoqrafiya:** Əl yazısı və daktilo görünüşü qatan xüsusi fontlar (`Caveat`, `Special Elite`, `Courier Prime`).

### 2. Çoxdilli Sistem (5 Dil)
Oyun 5 dildə tamamilə işləyir:
- 🇺🇸 İngiliscə
- 🇹🇷 Türkcə
- 🇦🇿 Azərbaycanca
- 🇷🇺 Rusca
- 🇪🇸 İspanca

### 3. Zəngin Söz Baza(ları)
JSON formatında xüsusi çətinliklərə bölünmüş söz lüğətləri.
- **Çətinlik dərəcələri:** Asan, Orta, Çətin, Kabus.

### 4. Aktiv və Gərgin Oyun Təcrübəsi
- **Sərt Şərhlər:** Offline "Savage AI", səhv etdiyiniz zaman sizə öz dilinizdə acımasız atmacalar atır.
- **Audio & Haptic:** Kağız cırılması, qələm sınması kimi real effektlər.

### 5. Statistika (SwiftData)
- Oynadığınız hər oyun `SwiftData` ilə yadda saxlanılır.
- Rəqabətə qoşulun və rekord seriyalarınızı artırın.

---

## 🚀 Proyekti Necə Çalışdırmaq Olar

1. Deponu klonlayın.
2. `Ink.xcodeproj` faylını **Xcode 15+** ilə açın.
3. **iOS 17.0+** cihazı və ya simulyatoru seçin.
4. **Cmd + R** düymələrinə vurun.
